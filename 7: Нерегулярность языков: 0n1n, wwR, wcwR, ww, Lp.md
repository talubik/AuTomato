Доказательство нерегулярности указанных языков с помощью леммы о накачке для регулярных языков.

Лемма о накачке (формулировка):
Для любого регулярного языка L существует константа p≥ 1 (длина накачки), такая что любое слово w ∈ L длиной |w| ≥ p можно разбить на три части w = xyz, удовлетворяющие условиям:

1. |xy| ≤ p
2. |y| ≥ 1
3. ∀k ≥ 0: xyᵏz ∈ L



1. Язык L₁ = {0ⁿ1ⁿ | n ≥ 0}

Доказательство:
Предположим,L₁ регулярен. Пусть p — константа из леммы о накачке.
Рассмотрим слово w= 0ᵖ1ᵖ, где |w| = 2p ≥ p.
Согласно лемме,w = xyz, где |xy| ≤ p и |y| ≥ 1.
Поскольку|xy| ≤ p, то xy состоит только из нулей: y = 0ᵐ, где m ≥ 1.
Рассмотрим слово w'= xy²z = 0ᵖ⁺ᵐ1ᵖ.
Так как m≥ 1, то количество нулей (p+m) не равно количеству единиц (p), значит w' ∉ L₁.
Противоречие.Язык нерегулярен.


2. Язык L₂ = {wwᴿ | w ∈ {0,1}*} (палиндромы чётной длины)

Доказательство:
Предположим,L₂ регулярен. Пусть p — константа из леммы.
Рассмотрим слово w= 0ᵖ110ᵖ. Заметим, что это слово имеет вид uuᴿ, где u = 0ᵖ1.
Согласно лемме,w = xyz, где |xy| ≤ p, |y| ≥ 1.
Поскольку|xy| ≤ p, то xy состоит только из первых p нулей: y = 0ᵐ, где m ≥ 1.
Рассмотрим слово w'= xy⁰z = xz = 0ᵖ⁻ᵐ110ᵖ.
Это слово имеет длину 2p- m + 2. Для того чтобы оно было палиндромом, его первая половина должна совпадать с реверсом второй половины. Но:

· Первые p-m символов: нули
· Следующие два символа: "11"
· Последние p символов: нули
  После реверса второй половины(0ᵖ11) мы получим 110ᵖ, что не совпадает с первой половиной 0ᵖ⁻ᵐ11 при m ≥ 1.
  Значит,w' ∉ L₂. Противоречие. Язык нерегулярен.


3. Язык L₃ = {wcwᴿ | w ∈ {0,1}*}

Доказательство:
Предположим,L₃ регулярен. Пусть p — константа.
Рассмотрим слово w= 0ᵖc0ᵖ. Здесь разделитель 'c' явно указывает середину.
Согласно лемме,w = xyz, где |xy| ≤ p, |y| ≥ 1.
Поскольку|xy| ≤ p, то xy находится в первой половине (до символа 'c'): y = 0ᵐ, где m ≥ 1.
Рассмотрим слово w'= xy²z = 0ᵖ⁺ᵐc0ᵖ.
Для принадлежности L₃ должно выполняться:часть до 'c' (0ᵖ⁺ᵐ) должна совпадать с реверсом части после 'c' (0ᵖ).
Но реверс 0ᵖ есть 0ᵖ,а 0ᵖ⁺ᵐ ≠ 0ᵖ при m ≥ 1.
Значит,w' ∉ L₃. Противоречие. Язык нерегулярен.


4. Язык L₄ = {ww | w ∈ {0,1}*} (слова, являющиеся двумя одинаковыми половинами)

Доказательство:
Предположим,L₄ регулярен. Пусть p — константа.
Рассмотрим слово w= 0ᵖ1ᵖ0ᵖ1ᵖ. Это слово имеет вид uu, где u = 0ᵖ1ᵖ.
Согласно лемме,w = xyz, где |xy| ≤ p, |y| ≥ 1.
Поскольку|xy| ≤ p, то xy состоит только из первых p нулей: y = 0ᵐ, где m ≥ 1.
Рассмотрим слово w'= xy²z = 0ᵖ⁺ᵐ1ᵖ0ᵖ1ᵖ.
Это слово должно быть представимо в виде vv для некоторого v.
Длина w'= 4p + m. Для чётности половины: |v| = (4p + m)/2 = 2p + m/2.
Поскольку m≥ 1, m/2 не обязательно целое, но даже если m чётно, сравним половинки:

· Первая "половинка" (предполагаемая): 0ᵖ⁺ᵐ1ᵖ
· Вторая "половинка": 0ᵖ1ᵖ
  Они не равны при m≥ 1. Значит, w' ∉ L₄. Противоречие. Язык нерегулярен.


5. Язык Lₚ = {aⁿ | n — простое число}

Доказательство:
Предположим,Lₚ регулярен. Пусть p — константа.
Выберем простое число n≥ p (такие числа существуют, простых чисел бесконечно много).
Рассмотрим слово w= aⁿ, где n — простое число.
Согласно лемме,w = xyz, где |xy| ≤ p, |y| ≥ 1.
Пусть|x| = i, |y| = m ≥ 1, |z| = j, причём i + m + j = n (простое число).
Рассмотрим слово w'= xyⁿ⁺¹z.
Длина w'= i + m(n+1) + j = i + m·n + m + j = (i + m + j) + m·n = n + m·n = n(m + 1).
Поскольку m≥ 1, то m+1 ≥ 2, и n(m+1) — составное число (имеет делители n и m+1).
Значит,w' ∉ Lₚ. Противоречие. Язык нерегулярен.


Вывод: Все пять языков являются нерегулярными, что доказано с помощью леммы о накачке для регулярных языков. Ключевой момент в каждом доказательстве — удачный выбор слова w, которое при "накачке" нарушает структуру, требуемую языком.
