0. Пролема соответствия Поста: *(Хопкрофт, стр. 401-412)*
	- Даны два конечных списка $A = (a_1 ... a_n)$ и $B = (b_1 ... b_n)$, где $a_i ∈ Σ^∗$ и $b_i ∈ Σ^∗$ для всех $i$. Вопрос существования непустой последовательности индексов $(i_1 \dots i_k)$, удовлетворяющей условию $a_{i_1} \dots a_{i_k} = b_{i_1} \dots b_{i_k}$, где $1 ⩽ i_j ⩽ n$ для всех $j$, называется **проблемой соответствий Поста (ПСП)**. Такую последовательность индексов, в случае её существования, называют **решением проблемы соответствий Поста**. 
	- Эта задача является одним из основных примеров неразрешимой задачи, использующийся для доказательства неразрешимости многих других задач.
1. Печатает ли программа *"hello, world"*, останавливается ли, вызывает ли определенную функцию, или выполняет любое другое нетривиальное действие - **неразрешимый вопрос**. В действительности для программ справдлив некий аналог теоремы Райса: всякое нетривиальное свойство программы, связанное с ее действиями *(но не лексическое или синтаксическое свойство самой программы)*, является неразрешимым *(Хопкрофт, стр. 413)*
2. Является ли данная КС-грамматика **неоднозначной**
	- Т.е. для какой-то цепочка языка можно построить два или более различных дерева вывода *(Хопкрофт, стр. 413-415)*
3. Проблема пустоты пересечения языков, порождаемых двумя КС-грамматиками. 
	- Даны две КС-грамматики $G_1$ и $G_2$ . Верно ли, что $L(G_1) \cap L(G_2)$?
4. 10-я проблема Гильберта. Существует ли универсальный алгоритм для определения, имеет ли произвольное диофантово уравнение целочисленные решения
	- (диофантово уравнение - уравнение вида многочлен $P(x_1 ... x_n) = 0$, где $P$ - многочлен с целочисленными коэффициентами)
	- (неразрешимость доказана в 1970г. Юрием Матиясевичем)
5. Проблема умирающей матрицы: для данного конечного множества квадратных матриц n × n определить, существует ли произведение всех или некоторых из этих матриц (возможно, с повторениями) в каком-либо порядке, дающее нулевую матрицу
	- Проблема неразрешима даже для n=3 (разрешимость для n=2 является открытым вопросом).
6. Проблема единичной матрицы: для данного конечного множества квадратных матриц n × n определить, существует ли произведение всех или некоторых из этих матриц (возможно, с повторениями) в каком-либо порядке, дающее единичную матрицу
	- Проблема неразрешима для целочисленных матриц начиная с n=4 и разрешима для n=2 (разрешимость для n=3 является открытым вопросом)
	- Проблема эквивалентна вопросу, является ли матричная полугруппа группой
