# Примеры проблем, неразрешимых

## 1. ДКА (детерминированные конечные автоматы)

**L = { ww | w ∈ {0,1}* }**  
ДКА не может запомнить первую половину строки для сравнения со второй.

**L = { 0^n 1^n | n ≥ 0 }**  
Требуется считать и сравнивать количество символов → нужна память, которой у ДКА нет.

---

## 2. МП-автоматы (магазинные автоматы, PDA)

**L = { 0^n 1^n 2^n | n ≥ 0 }**  
Один стек позволяет сравнить только две группы, но не три.

**L = { ww | w ∈ {0,1}* }**  
Требуется хранить произвольную строку целиком.

---

## 3. ДМП-автоматы, допускающие по заключительному состоянию  
(детерминированные PDA, принятие по финальному состоянию)

**L = { wcw^R | w ∈ {0,1}* }**  
Детерминированный PDA не может однозначно определить момент перехода от записи в стек к чтению из стека.

---

## 4. ДМП-автоматы, допускающие по пустому магазину  
(детерминированные PDA, принятие по пустому стеку)

**L = { a^i b^j c^k | i = j или j = k }**  
Требуется выбирать между двумя возможными стратегиями сопоставления, а детерминизм запрещает откат.

---

## 5. МТ, останавливающаяся на всех входах  
(тотальная машина Тьюринга)

**Проблема:**  
Дана МТ, которая гарантированно останавливается на всех входах.  
Вопрос: принимает ли она пустую строку?

**Неразрешима**, по сведению из общей проблемы останова и теореме Райса.

---

## 6. МТ (общая машина Тьюринга)

**Проблема останова:**  
Остановится ли машина Тьюринга на входе `w`?

**Неразрешима** (классический результат Тьюринга).

**Теорема Райса:**  
Любое нетривиальное свойство распознаваемого языка МТ неразрешимо  
(например, пустота языка, конечность, эквивалентность машин).
